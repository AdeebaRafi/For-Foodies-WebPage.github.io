# Foodie Delights

A simple website to show different food items like burgers, salads, rice dishes, and more. This project is made to help people see tasty food ideas with easy pictures and descriptions.

## Features

- Displays various food categories  
- Clean and easy design  
- Works on all devices  

## How to use

Open the `index.html` file in any web browser to see the website.

## About

This project is for sharing food ideas. It shows images and simple info about different meals. Good for beginners learning web design.
